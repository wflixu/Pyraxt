## Plugins

Axon is a versatile and extensible web framework that allows anyone to make plugins over the top of Axon.
Plugins in Axon allow you to enhance and customize the framework's functionality to suit your specific needs. Here are some noteworthy plugins that can supercharge your Axon-based projects:

### Rate Limit Plugin

- Description: This plugin enables you to implement rate limiting for your Axon application's routes. It helps prevent abuse, and brute-force attacks and ensures fair usage of your resources.
- GitHub repository: [axon-rate-limits](https://github.com/IdoKendo/axon_rate_limits)
- Installation:
  `python -m pip install axon-rate-limits`
- Usage:

```py
from axon import Axon, Request
from axon_rate_limits import InMemoryStore
from axon_rate_limits import RateLimiter

app = Axon(__file__)
limiter = RateLimiter(store=InMemoryStore, calls_limit=3, limit_ttl=100)

@app.before_request()
def middleware(request: Request):
    return limiter.handle_request(app, request)

@app.get("/")
def h():
    return "Hello, World!"

app.start(port=8080)
```

In this example, axon-rate-limits is used to enforce a rate limit of 3 requests per 100-seconds window for specific routes. If a client exceeds this limit, they will receive a "Too many requests" message.

The plugin integrates seamlessly with the Axon web framework, enhancing the security and stability of your application by preventing excessive requests from a single client.
