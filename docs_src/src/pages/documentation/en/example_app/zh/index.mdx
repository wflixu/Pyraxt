# 使用Axon构建实际应用

蝙蝠侠被委托开发一个Web应用程序来管理哥谭市的犯罪数据。该应用程序将允许哥谭警察局存储和检索有关犯罪活动、嫌疑人及其位置的数据。他决定使用Axon Web框架来高效快速地构建这个应用程序。

您可以在[这里](https://github.com/sparckles/example_axon_app)找到此应用程序的源代码。

## 安装Axon

第一步是安装Axon。蝙蝠侠创建了一个虚拟环境并使用pip安装了Axon。

```bash
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install axon
```

## 创建Axon应用程序

蝙蝠侠正准备创建一个`src/app.py`文件，这时他得知Axon提供了一个CLI工具来创建新应用程序。他运行以下命令来创建一个新的Axon应用程序：

```bash
$ python -m axon --create
```

这个示例应用程序将展示如何：

- 设计和实现RESTful API
- 处理身份验证和授权
- 使用中间件进行请求处理
- 实现实时通知
- 设置监控和日志记录
- 部署应用程序
- 生成API文档
- 组织和构建可维护的代码
