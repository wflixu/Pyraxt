## Installing Axon


Axon is a Python library that you can install using `pip` or `conda`

  <CodeGroup title="installation">

  ```bash {{ title: 'pip' }}
  pip install axon
  ```

  ```bash {{ title: 'conda' }}
  conda install axon -c conda-forge
  ```
  </CodeGroup>

While there are other more extensions of Axon like

  <CodeGroup title="installation">

  ```bash {{ title: 'pip' }}
  pip install "axon[templating]"
  ```

  ```bash {{ title: 'conda' }}
  conda install "axon[templating]" -c conda-forge
  ```
  </CodeGroup>



It is recommended to install the base package first and then install the extensions as needed.
