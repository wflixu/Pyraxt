## Redirection

Batman wanted to redirect some endpoints to others. Axon helped him do so by the following:

<Row>
  <Col>
  </Col>
  <Col sticky>
    <CodeGroup title="Request">

      ```python {{title: 'untyped'}}
      from axon import Axon, Response
      app = Axon(__file__)

      @app.get("/")
      async def index():
        return Response(
          status_code=307,
          description="",
          headers={"Location": "landing"},
        )

      @app.get("/landing")
      def landing():
        return "hii!"
      ```
    </CodeGroup>
  </Col>
</Row>
